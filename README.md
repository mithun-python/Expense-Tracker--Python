# Expense Tracker System (Python)

A simple file-based expense tracker application built using Python.
This project helps users record daily expenses and calculate total spending.

## Features
- Add daily expenses
- View all saved expenses
- Calculate total expense
- Data stored in CSV file
- Beginner friendly & customizable

## Technologies Used
- Python
- CSV Module
- File Handling


## How to Run
1. Install Python
2. Run the script:
```bash
python expense_tracker.py
